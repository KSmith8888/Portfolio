Restaurant - Admin Site

The admin website and API of a fictitious restuarant. Provides menu and event data that can be fetched by the separate public site and provides content management functionality to registered admins. User permissions are set by role, with admins having the ability to create, update and delete all types of content and employees only allowed to view information specific to them. Authorization is handled using JSON Web Tokens sent through secure, HttpOnly cookies and passwords are hashed using Bcrypt.

Made with NodeJS, MongoDB, HTML and CSS.

Bank Website

A website of a fictitious bank. Upon logging in, users are taken to an accounts overview page, where they can view account balances and transfer money between accounts. From there, they can navigate to pages for account details, displaying a list of transactions that can be filtered by type. On the account details pages, a credit score and report is also generated and displayed using a customized meter element. Other pages include a mortgage calculator, which fetches interest rate data from the U.S. Treasury API to calculate mortgage payment data based on amounts entered by the user and a products page displaying information about offered projects in a rotating carousel.

Made with JavaScript, HTML and CSS.

Requests are first sent though a proxy server (Proxy Server Project) to ensure that the API key is not sent to the client and that usage is limited. Built with the Parcel bundler.

BlackJack Browser Game

A classic BlackJack game, playable in the browser. The player and dealer receive cards that have been randomly selected from a generated deck and whoever gets closest to twenty-one without going over wins. Current scores and total money remaining each hand are displayed on the board. On subsequent hands, the all time high score is shown, which is stored in localStorage.

Made with TypeScript. HTML and CSS.
